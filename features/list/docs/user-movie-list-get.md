# ユーザー映画リスト取得

## 要件

### Why

- トップ画面と一覧画面で、ユーザーの登録済み作品を表示するためのデータを提供する。
- 作品詳細（TMDB連携情報）を保持している場合、表示に必要な構造へ整形して返す。

### What

- システムは指定されたリストに登録された作品一覧を取得できる。
- システムは配信サービス情報と作品詳細情報を結合して返せる。
- システムは監督情報を作品単位で集約できる。

### 受け入れ条件

- 有効なリスト指定に対して、登録済み作品の配列が返る。
- TMDB詳細が欠損している作品も、基本情報付きで返る。
- 作品が存在しないリストでは空配列を返す。

## 仕様

### How

- 取得
  - 登録作品を起点に、配信サービス情報と映画詳細情報を結合して取得する。
  - 取得順は新しい登録順（降順）とする。
- 監督情報
  - 作品詳細に紐づく映画情報を抽出し、監督情報を作品単位で集約する。
- 整形
  - 詳細情報が揃っている作品は、詳細付きの形式で返す。
  - 詳細情報が欠損している作品は、基本情報のみで返す。
  - 外部映画データベースの作品番号は数値に正規化し、変換不能な場合は内部の映画識別子を代替値として使用する。

### 失敗時挙動

- 本処理は内部で例外ハンドリングを持たないため、DB障害時は呼び出し元へ例外が伝播する。
