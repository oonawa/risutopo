# リストアイテム削除

## 要件

### Why

- ユーザーが不要になった作品をリストから除外できるようにする。
- リストの内容をユーザー操作に追随して最新状態に保つ。

### What

- システムは指定された登録作品をリストから削除できる。
- システムは削除対象の存在確認を行える。
- システムは削除結果を成功/失敗で返せる。

### 受け入れ条件

- 既に登録されている作品を指定した場合、削除が成功する。
- 登録されていない作品を指定した場合、失敗を返す。
- 削除後は同じ登録作品を再取得できない。

## 仕様

### How

- 入力
  - 削除対象の作品識別子を受け取る。
- 存在確認
  - 受け取った識別子と一致する登録作品が存在するかを確認する。
- 削除
  - 対象が存在する場合のみ削除処理を実行する。
- 応答
  - 正常終了時は成功レスポンスを返す。

### 失敗時挙動

- 対象未存在: 失敗を返す（`作品がリストへ登録されていないか、すでに削除されています。`）。
- DB例外: 失敗を返す（`映画の削除に失敗しました。`）。
